# Skill: Memoria Persistente

## 🎯 Propósito
Guardar y reutilizar información de ejecuciones anteriores.

---

## 📥 Cuándo usar
- Problemas repetidos
- Decisiones importantes
- Errores relevantes

---

## ⚙️ Reglas

### Guardar en:
- /memory/decisiones/
- /memory/patrones/
- /memory/errores/

---

## 🧠 Lógica
- Si ya existe solución → reutilizar
- Si no → guardar nueva
